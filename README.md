# 🚀 E-Commerce Web Application

A modern full-stack E-Commerce Web Application built using React, Vite, Tailwind CSS, Redux Toolkit, JWT Authentication, and Express mock backend.

🔗 **Live Demo:**  
https://ecom-webapplication.onrender.com/

---

# 📌 Features

## ✅ Product Management
- Product Listing
- Product Details Page
- Add Product
- Edit Product
- Delete Product
- Dynamic Product Rendering

## ✅ Shopping Cart
- Add to Cart
- Remove from Cart
- Quantity Management
- Dynamic Price Calculation
- Tax & Total Calculation

## ✅ Coupon System
- Coupon Validation
- Instant Discount Application
- Dynamic Payable Amount Updates

## ✅ Authentication
- Login & Signup
- JWT-Based Authentication
- Protected Routes
- Session Persistence using localStorage

## ✅ Checkout
- Order Summary
- GST/VAT Calculation
- Dynamic Billing

## ✅ UI/UX
- Fully Responsive Design
- Tailwind CSS Styling
- Modern Ecommerce UI
- Mobile Friendly
- Toast Notifications
- Reusable Components

# ⚙️ Installation & Setup


---

## 2️⃣ Install Dependencies

```bash

```
